  1. A design-oriented, simple YouTube client for Mac.

Youpp

Designed for Mac OSX, Youpp is built to watch YouTube videos as a floating window, allowing you to use other applications while watching.

Keyboard Shortcuts

⌘-F or Ctrl-F Toggles window floating. Sadly, in Windows, there is no button for this.

Youpp is an experimental and roughly written application with no goals for it to accomplish. But you'll have an awesome ad-free (except for in-video ads) experience :)

Warnings

Does not work yet in Windows. Windows build available! Downlod it now. Help fix the bugs and send a pull request!

Technical specs

Running on node-webkit without the need for servers, Youpp can be easily modified without setting up an IDE. Feel free to fork on GitHub!

Have node-webkit on your machine already? Download the .nw file directly.
